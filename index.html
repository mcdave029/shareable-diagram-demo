Architecture map — flowing left to right

Edge & Network

VPC with public/private subnets. ALB for HTTP(S); optional NLB for TCP/GRPC. Security groups + NACLs shape ingress.

Control Plane

EKS managed control plane in private subnets. API endpoints secured by IAM + RBAC.

Compute Pools

Managed node groups (EC2) and/or Fargate profiles. Separate pools for web/api vs jobs/workers.

Services & Mesh

LoadBalancer Services for web/api and grpc/tcp. Optional service mesh (if needed) layered on top.

Platform

VPC CNI for pod networking. CSI for EBS/FSx/GP3. IRSA for per-pod AWS creds. Autoscaling hooks as needed.

Data Plane

RDS/Aurora for relational. SQS/SNS/Kafka for async. S3/R2/B2 for objects. Observability plumbed to CloudWatch/Grafana.
